# Number Guessing Game (CLI)

A Python CLI number guessing game with difficulty levels, limited attempts, hints, dynamic range feedback, scoring, and auto-progression (Easy → Medium → Hard).

## Run

```bash
python game.py
```

From Windows PowerShell inside the `Number Guessing Game` folder:

```powershell
python .\game.py
```

## Features

- Difficulty levels: Easy, Medium, Hard
- Limited attempts per level
- Hints: hot/warm/cold on Easy/Medium
- Dynamic range hint updates after each guess
- Scoring system with bonus for remaining attempts
- Auto progression when you win a round
- Retry or change level on loss
- Quit anytime by typing `q`

Enjoy!
